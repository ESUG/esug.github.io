WebST: Web Components with PharoJS

Noury BOURAQADI (https://nootrix.com/)

Abstract: Web Components are standard web technologies that support the definition of reusable custom HTML elements. This JavaScript API allows defining both the view of web components as well as their behavior. Web component custom tags can encapsulate the view in a "shadow" DOM tree. This support creating web pages combining different web components from different sources while avoiding collisions.

This talk introduces WebST, a PharoJS-based solution to seamlessly implement and test web components using the powerful Pharo Smalltalk IDE. By delving into practical examples, we demonstrate how WebST facilitates the implementation and reuse of web components. Moreover, for production purposes, WebST generates web clients with standalone HTML and JavaScript code.
